# NumberSheet Project

A simple Java project for learning and experimentation.  
Generates random numbers, performs functional queries, and creates PDF reports.

## Features
- Number generation
- Functional queries (filter, map, reduce)
- Statistics (min, max, sum, average)
- BigInteger operations
- PDF report creation
- Configurable settings

## Purpose
This project is made for **learning**, **practice**, and **experimentation**.  
It is **free to modify**, extend, or reuse in any way.

## Technologies
- Java
- Maven
- PDFBox
- Stream API
- BigInteger

## How to Run
Use the main entry point:
ProgramLauncher.start();
## Notes
This project is not final or production-ready.  
It is a flexible base for studying Java concepts and building new ideas.
